# enrichment/common-errors/ — Role Placeholder

**Status:** Future-use placeholder. Not yet activated.

This directory is reserved for **common-error enrichment files** — supplementary
annotation records that extend the misconception nodes in
`knowledge/knowledge-map/misconceptions/` with additional real-world error
patterns observed in learner responses.

Distinguished from misconception nodes: misconception nodes are part of the
knowledge graph (Tier 4, graph-eligible). Common-error files here would be
enrichment metadata _about_ those nodes, not the nodes themselves.

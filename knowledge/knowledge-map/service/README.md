# knowledge-map / service

## Purpose

Contains knowledge records related to wine service, covered in RA1 of the WSET
Level 3 curriculum. This includes storage conditions, serving temperatures,
glassware, decanting, fault identification at service, and food and wine pairing.

## Schema

Service records follow a local schema (to be formalised). Key fields:
- `service_topic_id`
- `service_topic_name`
- `ra` — always `RA1`
- `key_principles`
- `common_errors`
- `related_concepts`
- `exam_question_angles`

## Naming convention

`{topic_slug}.json`  — e.g. `serving_temperature.json`, `decanting.json`
